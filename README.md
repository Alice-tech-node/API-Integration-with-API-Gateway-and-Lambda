# API-INTEGRATION WITH API GATEWAY AND LAMBDA.

Project Overview

This project exposed a Lambda function to the internet using Amazon API Gateway.

Objectives

 . Create a public API

 . Integrate Lambda with API Gateway

Services Used

 . Amazon API Gateway

 . AWS Lambda

Implementation Summary

An HTTP API was created and linked to a Lambda function, producing a publicly accessible endpoint.

Results & Outcomes

The API successfully returned responses from Lambda.

Skills Gained

 . API creation

 . Service integration
